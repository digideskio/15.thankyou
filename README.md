# 15.thankyou

A static site generator that uses [stasis][1] for the content and 
[broccoli][2] for the assets.

## Features

## Install

    gem install stasis

    cd nukomeet.com
    bower install

    npm install


## Usage

    cd nukomeet.com 
    npm start

## Build

    cd nukomeet.com
    npm run build

    cd dist
    python -m SimpleHTTPServer 8081

Open `localhost:8081`


[1]: http://stasis.me/ 
[2]: https://github.com/broccolijs/broccoli
